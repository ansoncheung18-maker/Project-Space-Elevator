# 核聚變飛船專案 Phase A：概念設計與可行性論證

**專案名稱：Project Fusion Spaceship**  
**版本：1.0**  
**更新日期：2026-06-22**  
**核心目標：設計一艘以核聚變反應爐為動力嘅星際飛船，實現快速、低成本嘅太陽系運輸。**

---

## 1. 願景與目標

> 化學火箭已經將人類送上月球，但要去火星需要 6-9 個月。核聚變飛船可以將呢個時間縮短到 30-45 天，徹底改變人類探索太陽系嘅能力。

### 1.1 核心目標

- 設計一艘以 **500 MW 核聚變反應爐**（Project Helios）為動力嘅飛船
- 地球 → 火星：**30-45 天**（化學火箭嘅 1/5）
- 載重：**200-500 噸**
- 比衝 (Isp)：**5,000-10,000 秒**（化學火箭嘅 10-20 倍）
- 燃料：氘（從海水提取，成本極低）

---

## 2. 點解核聚變飛船？

| 對比項 | 化學火箭 (Starship) | 核聚變飛船 |
|:---|:---|:---|
| 比衝 (Isp) | ~350 秒 | **5,000-10,000 秒** |
| 地球 → 火星 | 6-9 個月 | **30-45 天** |
| 燃料成本 | 高（每次數百萬美元） | **極低（氘，海水提取）** |
| 載重 | 100-150 噸 | **200-500 噸** |
| 可重複使用 | 部分 | **完全** |
| 技術成熟度 | ✅ 已成熟 | ⚠️ 需研發（基於 Project Helios） |

**結論：核聚變飛船喺速度、成本、載重上都優於化學火箭。**

### 3.2 推進原理

1. 核聚變反應爐產生高能等離子體（1 億°C）
2. 等離子體通過磁噴嘴加速排出
3. 產生推力（牛頓第三定律）
4. 比衝 5,000-10,000 秒，效率極高

## 7. 專家質疑與回應

### Q1：核聚變反應爐太重，點樣裝上飛船？
**A**：Project Helios 採用模塊化設計，反應爐核心質量約 50 噸。對比化學火箭引擎（約 10-20 噸），增加 30-40 噸質量換取 20 倍嘅效率提升，係合理取捨。

### Q2：輻射屏蔽太重，會唔會影響性能？
**A**：採用「水屏蔽」設計，水同時作為飲用水、冷卻劑同屏蔽層，唔係「額外」質量。400 cm 水屏蔽 + 8 cm 鉛屏蔽，總質量約 490 噸，但可確保船員安全。

### Q3：散熱板 10,454 m² 點樣放入火箭？
**A**：採用摺疊式設計，發射時收納（約 10m x 10m x 10m），進入太空後展開。類似 James Webb 太空望遠鏡嘅太陽罩展開技術，已有先例。

### Q4：氘燃料喺太空點樣儲存？
**A**：氘以液態儲存於低溫燃料艙（約 -250°C）。技術類似現有液氫儲存，已有成熟解決方案。

### Q5：點解唔直接用化學火箭？
**A**：化學火箭去火星要 6-9 個月，核聚變飛船只需 45 天。縮短航行時間可減少太空人輻射暴露、補給需求同心理壓力，係載人星際任務嘅關鍵。
