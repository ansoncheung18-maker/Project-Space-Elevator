# 核聚變飛船專案 Phase A：概念設計與可行性論證

**專案名稱：Project Fusion Spaceship**  
**版本：1.0**  
**更新日期：2026-06-22**  
**核心目標：設計一艘以核聚變反應爐為動力嘅星際飛船，實現快速、低成本嘅太陽系運輸。**

---

## 1. 願景與目標

> 化學火箭已經將人類送上月球，但要去火星需要 6-9 個月。核聚變飛船可以將呢個時間縮短到 30-45 天，徹底改變人類探索太陽系嘅能力。

### 1.1 核心目標

- 設計一艘以 **500 MW 核聚變反應爐**（Project Helios）為動力嘅飛船
- 地球 → 火星：**30-45 天**（化學火箭嘅 1/5）
- 載重：**200-500 噸**
- 比衝 (Isp)：**5,000-10,000 秒**（化學火箭嘅 10-20 倍）
- 燃料：氘（從海水提取，成本極低）

---

## 2. 點解核聚變飛船？

| 對比項 | 化學火箭 (Starship) | 核聚變飛船 |
|:---|:---|:---|
| 比衝 (Isp) | ~350 秒 | **5,000-10,000 秒** |
| 地球 → 火星 | 6-9 個月 | **30-45 天** |
| 燃料成本 | 高（每次數百萬美元） | **極低（氘，海水提取）** |
| 載重 | 100-150 噸 | **200-500 噸** |
| 可重複使用 | 部分 | **完全** |
| 技術成熟度 | ✅ 已成熟 | ⚠️ 需研發（基於 Project Helios） |

**結論：核聚變飛船喺速度、成本、載重上都優於化學火箭。**

---

## 3. 技術架構

### 3.1 飛船結構
貨物艙 / 載人艙（200-500 噸）       │
├─────────────────────────────────────┤
│  核聚變反應爐（500 MW）              │
│  - 引用 Project Helios               │
├─────────────────────────────────────┤
│  燃料艙（氘）                        │
├─────────────────────────────────────┤
│  磁噴嘴（推進系統）                  │
│  - 將等離子體加速排出                │
├─────────────────────────────────────┤
│  散熱系統                            │
│  - 大面積散熱板                      │
└─────────────────────────────────────┘

### 3.2 推進原理

1. 核聚變反應爐產生高能等離子體（1 億°C）
2. 等離子體通過磁噴嘴加速排出
3. 產生推力（牛頓第三定律）
4. 比衝 5,000-10,000 秒，效率極高

---

## 4. 可行性分析

### 4.1 科學可行性

| 條件 | 狀態 | 說明 |
|:---|:---|:---|
| 核聚變點火 | ✅ 已驗證 | 引用 Project Helios |
| 等離子體約束 | ✅ 已驗證 | 超導磁體技術 |
| 磁噴嘴推進 | ✅ 理論可行 | 已有實驗驗證 |
| 太空環境 | ✅ 有利 | 真空、低溫有助於聚變 |

### 4.2 工程可行性

| 挑戰 | 解決方案 | 時間 |
|:---|:---|:---|
| 反應爐小型化 | 採用模塊化設計 | 5-10 年 |
| 輻射屏蔽 | 輕量化屏蔽材料 | 3-5 年 |
| 散熱系統 | 大面積散熱板 | 2-3 年 |
| 推進系統整合 | 系統工程設計 | 3-5 年 |

### 4.3 經濟可行性

| 參數 | 數值 |
|:---|:---|
| 總投資（研發 + 建造） | $200-400 億 |
| 每趟燃料成本 | <$100 萬（氘） |
| 每公斤運費（地球 → 火星） | $200-500 |
| 年運量 | 2,000-5,000 噸 |
| 投資回收期 | 10-20 年 |

---

## 5. 與 Project Helios 嘅關係

| 項目 | Project Helios | 核聚變飛船 |
|:---|:---|:---|
| 反應爐功率 | 500 MW | 500 MW（相同） |
| 燃料 | 氘 | 氘（相同） |
| 用途 | 地面發電 | **太空推進** |
| 技術差異 | 熱能 → 電能 | **熱能 → 推力** |

**核聚變飛船本質上係將 Project Helios 嘅反應爐「裝上飛船」，並加上磁噴嘴。**

---

## 6. 結論

✅ 科學上可行（核聚變 + 磁噴嘴）  
✅ 工程上可行（分階段研發）  
✅ 經濟上可行（長期回報）  
✅ 戰略價值巨大（太陽系運輸革命）  

**核聚變飛船係人類邁向「太陽系文明」嘅關鍵一步。**
