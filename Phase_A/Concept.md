# 太空電梯專案 Phase A：概念設計與可行性論證

**專案名稱：Project Space Elevator**  
**版本：1.0**  
**更新日期：2026-06-02**  
**核心目標：建造人類首座太空電梯，將地球至同步軌道嘅運輸成本降低 99%。**

---

## 1. 願景與目標

> 太空電梯係人類通往宇宙嘅「天梯」。一旦建成，火箭將成為歷史，太陽系將成為人類嘅經濟腹地。

### 1.1 核心目標

- 建造一座連接地球赤道至地球靜止軌道 (GEO，約 35,786 公里) 嘅太空電梯
- 將每公斤貨物運送成本從火箭嘅 $10,000-50,000 降至 **$100-500**（降低 99%）
- 每日運量：100 噸貨物 + 人員
- 動力來源：核聚變反應爐（你嘅 Project Helios）

## 2. 技術架構

### 2.1 太空電梯四大組成

| 組件 | 描述 | 關鍵參數 |
|:---|:---|:---|
| **1. 錨點 (Anchor)** | 地面固定站，設於赤道海上平台 | 位置：西經 100° 赤道（太平洋） |
| **2. 纜索 (Tether)** | 連接地球至太空嘅核心結構 | 材料：碳納米管 / 石墨烯 |
| **3. 配重 (Counterweight)** | 位於纜索外端，用離心力拉直纜索 | 位置：GEO 以外 100,000 公里 |
| **4. 升降機 (Climber)** | 沿纜索爬升嘅「太空纜車」 | 動力：核聚變無線傳輸 |

## 2.6 選址分析：太空電梯連接邊度？

### 2.6.1 基本原則

太空電梯嘅地面錨點必須滿足三個條件：

| 條件 | 原因 | 最佳選擇 |
|:---|:---|:---|
| **位於赤道 (0° 緯度)** | 利用地球自轉最大離心力，減少纜索拉力 | 赤道線 |
| **海洋上** | 避免墜落時擊中城市，方便移動避開風暴 | 太平洋、大西洋、印度洋 |
| **遠離人口密集區** | 安全考量 | 遠離陸地 |

### 2.6.2 三個潛在選址

| 選址 | 位置 | 優點 | 缺點 |
|:---|:---|:---|:---|
| **方案 A：西經 100° 赤道（太平洋）** | 南美洲以西，加拉帕戈斯群島附近 | 遠離陸地、風暴較少 | 距離美國/亞洲較遠 |
| **方案 B：東經 110° 赤道（印度洋）** | 印尼以西 | 靠近亞洲市場 | 風暴較多、印尼地震帶 |
| **方案 C：西經 0° 赤道（大西洋）** | 非洲以西，巴西以東 | 靠近歐美 | 風暴較多、船運繁忙 |

### 2.6.3 推薦方案

**推薦方案 A：西經 100° 赤道（太平洋）**

| 參數 | 數值 |
|:---|:---|
| 位置 | 0°N, 100°W |
| 最近陸地 | 南美洲（約 2,500 公里） |
| 水深 | 約 4,000 米 |
| 風暴頻率 | 低（遠離颱風路徑） |
| 錨點類型 | 海上移動平台 |

**理由**：
- 遠離陸地，墜落風險最低
- 風暴較少，操作環境穩定
- 水深適中，適合建造海上平台
- 可通過船運連接美洲、亞洲市場

### 2.6.4 配重位置

| 參數 | 數值 |
|:---|:---|
| 地球靜止軌道 (GEO) | 35,786 km |
| 配重位置 | **100,000 km**（GEO 以外） |
| 總纜索長度 | **約 100,000 km** |

**理由**：配重位於 100,000 km 處，可提供足夠離心力拉直纜索，同時避免與其他衛星碰撞。

### 2.6.5 連接嘅軌道位置

太空電梯嘅頂端可以連接多個「太空站」：

| 高度 | 用途 | 說明 |
|:---|:---|:---|
| **低地球軌道 (LEO，~500 km)** | 衛星部署、太空站補給 | 需中途站 |
| **中地球軌道 (MEO，~20,000 km)** | GPS 衛星、導航 | 可停靠 |
| **地球靜止軌道 (GEO，~35,786 km)** | 通訊衛星、電視廣播 | 主要目的地 |
| **配重端 (100,000 km)** | 深空任務出發點 | 前往月球、火星 |

✅ **太空電梯唔係只連接一個點，而係一條「垂直高速公路」**，升降機可以喺任何高度停靠。
