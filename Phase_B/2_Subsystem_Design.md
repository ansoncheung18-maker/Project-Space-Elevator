# 核聚變飛船專案 Phase B：子系統設計

**版本：1.0**  
**更新日期：2026-06-23**

---

## 1. 推進子系統

### 1.1 設計目標
- 將 500 MW 核聚變能量轉化為推力
- 比衝 >8,000 秒
- 推力 >50 kN

### 1.2 技術方案

| 組件 | 規格 | 說明 |
|:---|:---|:---|
| 核聚變反應爐 | 500 MW | 引用 Project Helios |
| 磁噴嘴 | 超導磁體 | 加速等離子體 |
| 燃料注入 | 氘氣體 | 連續注入 |
| 推力控制 | 調節燃料流量 | 精確控制 |

### 1.3 推進原理

1. 反應爐產生高能等離子體（1 億°C）
2. 等離子體通過磁噴嘴加速
3. 加速後嘅等離子體排出產生推力
4. 比衝 8,000 秒，效率極高

---

## 2. 輻射屏蔽子系統

### 2.1 設計目標
- 將中子通量降至安全水平（<1e4 n/m²/s）
- 屏蔽質量 <500 噸

### 2.2 技術方案

| 層數 | 材料 | 厚度 | 功能 |
|:---|:---|:---|:---|
| 第 1 層 | 水 | 400 cm | 吸收中子 |
| 第 2 層 | 鉛 | 8 cm | 阻擋伽馬 |

### 2.3 水屏蔽嘅優勢

- 氫含量高，有效吸收中子
- 同時作為飲用水同冷卻劑
- 無需額外運送質量

---

## 3. 散熱子系統

### 3.1 設計目標
- 散走 350 MW 廢熱
- 散熱板面積 <15,000 m²

### 3.2 技術方案

| 參數 | 數值 |
|:---|:---|
| 散熱板溫度 | 900 K |
| 散熱板面積 | 10,454 m² |
| 材料 | 碳碳複合材料 |
| 設計 | 摺疊式 |

### 3.3 散熱板展開

1. 發射時收納（10m x 10m x 10m）
2. 進入太空後展開
3. 類似 James Webb 太陽罩技術

---

## 4. 燃料儲存子系統

### 4.1 設計目標
- 儲存 300 噸氘燃料
- 保持液態（-250°C）

### 4.2 技術方案

| 參數 | 數值 |
|:---|:---|
| 燃料 | 氘（液態） |
| 儲存溫度 | -250°C |
| 儲存容器 | 雙層低溫艙 |
| 蒸發率 | <0.1%/日 |

### 4.3 技術來源

- 類似現有液氫儲存技術
- 已喺火箭上使用數十年
