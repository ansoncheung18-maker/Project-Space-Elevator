# 太空電梯專案 Phase A：概念設計與可行性論證

**專案名稱：Project Space Elevator**  
**版本：1.0**  
**更新日期：2026-06-02**  
**核心目標：建造人類首座太空電梯，將地球至同步軌道嘅運輸成本降低 99%。**

---

## 1. 願景與目標

> 太空電梯係人類通往宇宙嘅「天梯」。一旦建成，火箭將成為歷史，太陽系將成為人類嘅經濟腹地。

### 1.1 核心目標

- 建造一座連接地球赤道至地球靜止軌道 (GEO，約 35,786 公里) 嘅太空電梯
- 將每公斤貨物運送成本從火箭嘅 $10,000-50,000 降至 **$100-500**（降低 99%）
- 每日運量：100 噸貨物 + 人員
- 動力來源：核聚變反應爐（你嘅 Project Helios）

---

## 2. 技術架構

### 2.1 太空電梯四大組成

| 組件 | 描述 | 關鍵參數 |
|:---|:---|:---|
| **1. 錨點 (Anchor)** | 地面固定站，設於赤道海上平台 | 位置：西經 100° 赤道（太平洋） |
| **2. 纜索 (Tether)** | 連接地球至太空嘅核心結構 | 材料：碳納米管 / 石墨烯 |
| **3. 配重 (Counterweight)** | 位於纜索外端，用離心力拉直纜索 | 位置：GEO 以外 100,000 公里 |
| **4. 升降機 (Climber)** | 沿纜索爬升嘅「太空纜車」 | 動力：核聚變無線傳輸 |

### 2.2 纜索設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 長度 | 約 100,000 公里 | 從地面到配重 |
| 材料 | 碳納米管 (CNT) | 強度重量比 >50,000 kN·m/kg |
| 形狀 | 錐形帶狀 | 根部最粗，尖部最細 |
| 根部寬度 | 1 米 | 承受最大拉力 |
| 尖部寬度 | 1 毫米 | 配重端 |
| 厚度 | 1 毫米 | 帶狀 |

### 2.3 錨點設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 類型 | 海上移動平台 | 可避開風暴 |
| 位置 | 赤道（0° 緯度） | 利用地球自轉離心力 |
| 平台尺寸 | 1 km x 2 km | 跑道、貨運中心 |
| 固定方式 | 動態定位 + 壓載 | 抵禦風浪 |

### 2.4 配重設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 類型 | 捕獲嘅小行星 | 或 太空站 + 壓艙物 |
| 質量 | 約 1,000 噸 | 提供足夠離心力 |
| 位置 | GEO 以外 100,000 公里 | 保持纜索繃緊 |

### 2.5 升降機 (Climber) 設計

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 數量 | 10-20 台 | 同時運行 |
| 速度 | 200 公里/小時 | 至 GEO 需約 7.5 天 |
| 載重 | 10 噸/台 | 貨物 + 乘客 |
| 動力 | 核聚變無線傳輸 | 地面反應爐提供能源 |

## 3. 關鍵技術挑戰與解決方案

| 挑戰 | 說明 | 解決方案 | 狀態 |
|:---|:---|:---|:---|
| **材料強度** | 需要強度重量比 >50,000 嘅材料 | 碳納米管 / 石墨烯 | 實驗室可行，量產中 |
| **纜索製造** | 長度 100,000 公里，需太空製造 | 從 GEO 向下「生長」 | 需軌道工廠 |
| **動力傳輸** | 升降機需要大量能源 | 核聚變無線傳輸 | 你嘅 Project Helios |
| **太空垃圾** | 碰撞會切斷纜索 | 雷射清除 + 避讓系統 | 技術可行 |
| **天氣** | 雷擊、風暴 | 錨點可移動 + 避雷設計 | 工程可行 |
